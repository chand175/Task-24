What is javascript?
JavaScript is a programming language used primarily for web development. It adds interactivity and dynamic behavior to websites, allowing them to respond to user actions in real-time, such as clicking buttons, filling forms, or moving elements on the page. In simple words, JavaScript is what makes websites come alive and allows developers to create engaging and interactive experiences for users.
Full form of WWW?
WWW stands for World Wide Web. It's a system of interconnected documents and resources that are accessed over the internet through web browsers. The WWW was invented by Tim Berners-Lee in 1989 and has since become a fundamental component of the internet, enabling the sharing and accessing of information globally.
what is URL?
URL stands for Uniform Resource Locator. It's a reference or address used to locate resources on the internet. In simpler terms, it's the web address you type into your browser to access a specific webpage or resource.
